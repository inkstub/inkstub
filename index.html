# STUB — Collectible Ticket Keepsakes Turn digital tickets into physical, old-school collectible stubs. ## Project Structure ``` stub/ ├── index.html ← Main app (all views) ├── css/ │ └── style.css ← All styles + ticket themes ├── js/ │ ├── tickets.js ← Ticket HTML rendering │ └── app.js ← App logic, nav, orders └── README.md ``` ## Features (Current) - Upload e-ticket (file picker + drag & drop) - Manual form entry for event details - 3 vintage ticket styles: Classic, Gameday, Rock & Roll - Live ticket preview - Order flow with address collection - My Collection view with filter by event type - Order status badges (Saved, Printing, Shipped, Delivered) - Persists collection in localStorage ## How to Deploy (Free, in 5 minutes) ### Option A: Vercel (Recommended) 1. Create a free account at https://vercel.com 2. Create a free account at https://github.com 3. Create a new GitHub repo, upload these files 4. In Vercel → "Add New Project" → import your repo 5. Click Deploy — you're live with a .vercel.app URL ### Option B: Netlify 1. Go to https://netlify.com → Sign up free 2. Drag and drop this entire `stub/` folder onto the Netlify dashboard 3. You're live instantly — no GitHub required ### Option C: GitHub Pages 1. Push files to a GitHub repo 2. Settings → Pages → Source: main branch / root 3. Your site is live at https://yourusername.github.io/stub ## Next Steps to Make It Real 1. **Stripe** — Replace the fake order button with Stripe Checkout - https://stripe.com — create account, get API keys - Add a simple serverless function (Vercel/Netlify functions) to create a Stripe session 2. **Print API** — Connect to a print-on-demand service - Gelato API (https://gelato.com/api) — best for custom print products - Printful API (https://printful.com/api) — good alternative 3. **OCR / AI ticket reading** — Auto-fill form from uploaded ticket - Claude API with vision: send the uploaded image, get back structured JSON - Endpoint: https://api.anthropic.com/v1/messages 4. **User Accounts** — Let users log in and see orders - Supabase (https://supabase.com) — free tier, easy auth + database ## Local Development No build step needed. Just open `index.html` in a browser, or run: ```bash # Python (usually pre-installed) python3 -m http.server 3000 # Node npx serve . ``` Then visit http://localhost:3000
